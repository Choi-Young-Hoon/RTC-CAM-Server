# RTC-CAM

## RTC-CAM이란?
RTC-CAM은 WebRTC 기반의 익명 화상 채팅 서비스 입니다.

[RTC-CAM 바로가기](https://choiyh.synology.me:40001)


## 시작하기
### 1. RTC-CAM 설치하기

```git clone https://github.com/Choi-Young-Hoon/RTC-CAM-Server ```

``` go build ```

``` ./rtccam -g   # 기본 설정파일 생성 ```

``` vim config.yaml # 설정파일 수정 ```

``` ./rtccam # RTC-CAM 시작 ```

### 2. RTC-CAM 사용하기

``` http://localhost:40001 ```
